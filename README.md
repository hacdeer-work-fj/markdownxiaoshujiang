# MarkDown 小书匠 📚

> 个人知识笔记项目 · 用 Markdown 书写，用 Git 管理，用 Obsidian 沉淀

## 简介

「MarkDown 小书匠」是一个个人知识管理项目，专注用 Markdown 构建第二大脑。所有笔记从碎片化输入到系统化输出，遵循 **PARA 体系** 组织。

## 目录结构

```
markdownxiaoshujiang/
├── README.md               # 项目总览
├── .gitignore              # Git 忽略规则
├── Projects/               # 活跃项目 - 当前在做什么
│   └── ...
├── Areas/                  # 责任领域 - 长期负责什么
│   └── ...
├── Resources/              # 资源归档 - 未来可能用到的素材
│   └── ...
├── Archives/               # 冷存储 - 已完成或不再活跃
│   └── ...
└── Templates/              # 模板 - 可复用的笔记模板
    └── ...
```

## 使用工具

| 工具 | 用途 |
|------|------|
| **Obsidian** | 本地笔记编辑与知识图谱 |
| **Git + GitHub** | 版本控制与云端同步 |
| **Markdown** | 统一书写格式 |
| **Hermes Agent** | AI 辅助知识处理 |
